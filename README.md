## Description
It is a simple Laravel base application that allows you to keep and manage a directory of personal links.

## Demo
[You can browse application demo and play with dummy data.](https://bookmarks.monolitpro.info)

## Installation
1. Clone repository to your wordpress installation **wp-content/themes** folder
2. Go to cloned directory **cd wp-content/themes/wordpress_theme_with_autoload_boilerplate**
3. Execute composer command **composer dump-autoload**
